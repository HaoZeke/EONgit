
.. _dimer:

================================
Dimer
================================

Options for controlling the dimer minimum mode finding method on the client side, specified in the **[Dimer]** section of the config.ini file.

**improved**: Improvements to the dimer method from Kastner.

    type: boolean

    default: ``False``

**torque_min**: Minimum torque above which the dimer rotates only once and below which is does not rotate.

    type: float

    default: ``0.1``

**opt_method**: Optimization algorithm to choose the dimer rotation direction

    type: string

    default: ``sd``

    options:

        ``sd``: steepest descent, rotate along the rotational force.

        ``cg``: conjudate gradient, rotate along conjugate directions.

        ``lbfgs``: quasi-Newton method [not implemeted yet].

**torque_max**: Maximum torque above which the dimer rotates up to rotations_max times.

    type: float

    default: ``1.0``

**finite_angle**: Finite difference angle over which the dimer is rotated to find the lowest curvature.

    type: float

    default: ``0.005``

**rotations_max**: Maximum number of rotations allowed for the dimer in each step.

    type: int

    default: ``8``

**rotations_min**: Minimum number of rotations allowed for the dimer in each step.

    type: int

    default: ``1``

**converged_angle**: Dimer is considered converged if it rotates fewer degrees than this.

    type: float

    default: ``5.0``


.. _lanczos:

================================
Lanczos
================================

Options for controlling the Lanczos minimum mode finding method on the client side.

**tolerance**: This is the convergence critera for relative error of the lowest eigenvalue.

    type: float

    default: ``0.001``

**max_iterations**: The maximum number of refinement iterations when calculating the minimum eigenvalue.

    type: int

    default: ``20``

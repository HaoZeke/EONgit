
.. _download:

================================
Download
================================

EON is released under the `GNU General Public License version 3`_.

The code can be anonymously checked out of the subversion repository with the
following command:

.. code-block:: sh

    svn co http://theory.cm.utexas.edu/svn/eon

This will fetch a copy of the latest code to a local directory named eon. 
For more information on subversion read its `online documentation`_.

The latest revision can also be downloaded as a gzipped tar file: eon-trunk.tgz_

.. _online documentation: http://svnbook.red-bean.com/en/1.5/index.html

.. _eon-trunk.tgz: http://theory.cm.utexas.edu/eon-files/eon-trunk.tgz
